# 🏆 隊伍分數計分板 (Scoreboard)

這是一個簡單且即時同步的網頁版隊伍計分系統。設計用於活動、比賽或遊戲中，讓主持人可以透過後台控制分數，而觀眾透過前台投影畫面即時看到最新戰況。

## ✨ 特色

- **雙螢幕設計**：
  - **顯示畫面 (`index.html`)**：純淨的展示介面，適合投影到大螢幕。
  - **控制畫面 (`control.html`)**：操作介面，可調整分數、修改隊名與更換圖片。
- **即時同步**：利用瀏覽器的 `localStorage` 技術，兩個視窗之間的資料會自動同步（需在同一個瀏覽器開啟）。
- **圖片客製化**：點擊隊伍圖片即可上傳更換照片（暫存於瀏覽器中）。
- **分數控制**：
  - 支援快速加減分：每次點擊 **+** 或 **-** 按鈕，分數會以 **10 分** 為單位增減。
  - **一鍵歸零**：提供「清空所有分數」按鈕，可快速將所有隊伍分數重置為 0（保留隊名與圖片）。
- **隊名編輯**：可隨時修改隊伍名稱。

## 🚀 如何使用

1. **開啟顯示畫面**：
   - 雙擊 `index.html`。
   - 將此視窗拖曳到投影機或第二螢幕，並切換至全螢幕 (F11)。

2. **開啟控制畫面**：
   - 雙擊 `control.html`。
   - 保留此視窗在您的電腦螢幕上。

3. **開始操作**：
   - **調整分數**：點擊 `+` 增加 10 分，點擊 `-` 減少 10 分。
   - **重置比賽**：點擊上方的「🔄 清空所有分數」按鈕，確認後即可將分數歸零。
   - **更換圖片**：點擊隊伍圖片可上傳新照片。
   - **修改隊名**：在輸入框中修改隊伍名稱。
   - 觀察顯示畫面，內容會自動同步更新！

## 🛠️ 技術說明

- **前端技術**：HTML5, CSS3, JavaScript (Vanilla JS)。
- **資料儲存**：使用 `localStorage` 儲存隊伍資訊（名稱、分數、圖片 Base64）。
- **同步機制**：
  - `control.html` 修改資料後寫入 `localStorage`。
  - `index.html` 每 0.5 秒輪詢 `localStorage` 檢查是否有更新並重新渲染。

## ⚠️ 注意事項

- 圖片與設定僅暫存於您的瀏覽器中，清除瀏覽器快取可能會重置資料。
- 請確保兩個檔案都在**同一個瀏覽器**（例如都是 Chrome 或都是 Edge）中開啟，否則無法同步。
